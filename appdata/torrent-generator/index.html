Quick Import Guide

  1. Create: Drop your file, pick a piece size, click Generate .torrent. All hashing is done locally in your browser — nothing is uploaded anywhere.
  2. Import to qBittorrent: Add the .torrent to qBittorrent on your server. Set the save path to the directory that already contains the file (e.g. /data/torrents/movies/). qBittorrent will fast-resume instantly since the file already exists.
  3. Prowlarr integration: Add a Generic Torznab indexer in Prowlarr pointing to http://media_local_tracker:8000/api. This propagates to Radarr/Sonarr for import coordination.
  4. LAN clients: The LAN Announce URL (auto-filled from your server's IP) is baked into the torrent alongside the Docker-internal URL, so both work simultaneously.
