Installing from scratch will **NUKE and completely delete** any existing configurations, databases, and container states for the selected services. This resets the service to its clean, first-time state. Raw external media files (/mnt/hdd etc.) will **NOT** be touched.

Install Docker Engine

Install Docker Engine and Docker Compose V2 on your Linux server

🐳 Docker is Required

Docker Engine and Docker Compose V2 must be installed before running the homeserver stack. This page runs the built-in installer from setup.sh --install-docker. The script uses the official get.docker.com convenience installer.

1

Checks for existing Docker

The script first checks whether docker is already present on the system. If so, it reports the version and skips reinstallation.

2

Installs via official convenience script

If Docker is missing, the installer downloads and runs the official curl -fsSL https://get.docker.com | sh script — the recommended method for Ubuntu, Debian, Fedora, and CentOS.

3

Enables Docker service & installs Compose V2

Runs systemctl enable --now docker and installs the docker-compose-plugin via your system's package manager so docker compose is available.

4

Verifies installation

Confirms both docker --version and docker compose version succeed before completing. Any errors are surfaced in the console below.

⚠️ Linux Only

Automatic installation via this portal is only supported on Linux (Ubuntu/Debian/Fedora/CentOS). On macOS, please install Docker Desktop manually. The script must be run with sudo / root privileges on the server.

Tailscale Secure Remote Access

            

              Tailscale connects your homeserver to other devices using an encrypted WireGuard VPN mesh (called a Tailnet). This lets you manage the server and use its apps from anywhere in the world without opening ports on your home router.
            

            

              

                
1

                

                  
Generate Auth Token

                  
Log into the Tailscale Admin Console. Go to Settings > Keys and click Generate Auth Key. Copy the key (starts with tskey-auth-).

                

              

              

                
2

                

                  
Configure & Start Stack

                  
Navigate to the Configure Tailscale VPN page in this portal. Paste your key into the Auth Key input field and click Start Tailscale Stack. The server will launch the container and join your private Tailnet.

                

              

              

                
3

                

                  
Connect Your Client Devices

                  
Install the Tailscale client on your phone, laptop, or tablet. Log in using the same Tailscale account. You will immediately see your homeserver listed as a connected node with its own secure 100.x.x.x IP address. Enter that IP in your browser to access the server remotely!

                

              

            

          


          
          

            
Nextcloud Cloud Sync & Storage

            

              Nextcloud provides a complete cloud replacement on your own server, letting you sync folders, view office files, track tasks, and sync calendars.
            

            

              

                
1

                

                  
Create Admin Account

                  
Open Nextcloud in your browser at http://[SERVER_IP]:8080. On the initial startup screen, enter a secure username and password to create the Administrator account.

                

              

              

                
2

                

                  
Install Desktop & Mobile Clients

                  
Download the Nextcloud Client app on your desktop (Windows/macOS/Linux) or mobile phone (iOS/Android). Open the app and choose "Log In".

                

              

              

                
3

                

                  
Connect & Sync

                  
Enter your server URL http://[SERVER_IP]:8080 (or your Tailscale 100.x.x.x IP if remote) and log in with your credentials. Choose which folders on your machine to sync to your private cloud storage.

                

              

            

          


          
          

            
Immich Photo & Video Backup

Create Account & Get Mobile App

Open the mobile app and connect using the server endpoint http://[SERVER_IP]:2283/api. Log in, select your camera roll albums, and tap Enable Backup to upload photos automatically in the background!

                

              

            

          


          
          

            
Jellyfin Media Library Streaming

            

              Jellyfin indexes movies, shows, and music stored in your media directory. It fetches metadata/posters automatically and streams them smoothly to TVs, phones, or web browsers.
            

            

              

                
1

                

                  
Organize Media Paths

                  
Store your video files inside your server media folder (e.g., /mnt/hdd/movies or ./data/media). Jellyfin will scan these folders to index them into your virtual library.

                

              

              

                
2

                

                  
Initial Library Setup

                  
Open Jellyfin in your browser at http://[SERVER_IP]:8096. Follow the initial wizard, choose a language, and click Add Media Library. Map the content type (e.g. Movies) to the host folder path.

                

              

              

                
3

                

                  
Connect to Clients (TVs, Phones)

                  
Install the Jellyfin App on Fire TV, Apple TV, Roku, or iOS/Android devices. Connect to server http://[SERVER_IP]:8096 and log in to stream your high-definition media library immediately!
